## 项目结构

```
bingomini/
├── index.html   # 主页面（单页多区块）
├── styles.css   # 样式
```

## 本地预览

```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js
npx serve .
```

访问 http://localhost:8080
